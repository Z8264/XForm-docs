# schema

## 通用配置

| 名称       | 类型     | 默认值 | 描述                              |
| ---------- | -------- | ------ | --------------------------------- |
| c          | `string` | `-`    | 组件名称                          |
| xs         | `number` | 12     | 占据网格的空间大小，可选值：1～12 |
| default    | `any`    | `''`   | 默认值                            |
| label      | `string` | `''`   | 组件标题                          |
| helperText | `string` | `''`   | 提示信息                          |
| rule       | `func`   | `-`    | 组件校验规则                      |

## 组件状态

| 名称     | 类型                 | 默认值  | 描述     |
| -------- | -------------------- | ------- | -------- |
| visible  | `func` &#124; `bool` | `true`  | 显示     |
| disabled | `func` &#124; `bool` | `false` | 禁止编辑 |
